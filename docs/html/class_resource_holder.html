Detailed Description

template<typename Resource, typename Identifier>
class ResourceHolder< Resource, Identifier >

Abstract class to load and hold resources in memory like textures, fonts and sounds.

ResourceHolder is a template for classes that manage external resources like images or sound. It has methods to load resources from a path and to get previously loaded resources with an identifier. Types using this template prepared for use for all used resources are defined in resource_identifiers.h

Template Parameters
ResourceThe type of resource to load, for example, sf::Texture.
Oneof the identifier types defined in resource_identifiers.h
